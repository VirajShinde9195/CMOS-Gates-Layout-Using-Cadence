# CMOS Gates Layout Using Cadence

## Overview
This repository contains schematic and layout designs of CMOS logic gates created using Cadence Virtuoso.  
The project demonstrates transistor-level implementation of digital circuits using CMOS technology and VLSI design flow.

## Contents
- CMOS Inverter
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate
- Transmission Gate
- Multiplexer Designs
- Schematic Designs
- Layout Designs

## Tools Used
- Cadence Virtuoso
- CMOS VLSI Design Concepts

## Features
- Transistor-level CMOS circuit design
- Pull-Up Network (PUN) and Pull-Down Network (PDN)
- Schematic and layout implementation
- Layout design following CMOS design rules
- Basic VLSI physical design flow

## Design Flow
1. Schematic Design
2. Circuit Simulation
3. Layout Creation
4. DRC (Design Rule Check)
5. LVS (Layout Versus Schematic)

## Learning Outcomes
- CMOS digital circuit design
- Schematic to layout conversion
- Layout optimization techniques
- Understanding DRC and LVS verification
- Practical exposure to Cadence Virtuoso

## Applications
- Digital IC Design
- VLSI Physical Design
- CMOS Standard Cell Design
- Semiconductor Circuit Design
